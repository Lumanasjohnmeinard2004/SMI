# SMI Cooperative System

The **SMI Cooperative System** contains the following project components:

- `smi-frontend` – Frontend application
- `smi-backend` – Backend server and API
- `smi_coop_db.backup` – PostgreSQL database backup file

## Database Setup

This project uses **PostgreSQL** as its database.

To test the system locally:

1. Install PostgreSQL and pgAdmin.
2. Create a new PostgreSQL database named:

   ```text
   smi_coop_db
